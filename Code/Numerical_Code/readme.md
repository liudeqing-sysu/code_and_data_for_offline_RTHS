# Numerical Code

This folder contains the numerical benchmark and post-processing scripts for
the offline RTHS study.

- `adaptive_relaxation_coupling.py`: ARFI/Aitken coupling benchmark.
- `displacement_driven_beam.py`: Dirichlet beam substructure solver.
- `force_driven_beam.py`: Neumann beam substructure solver.
- `dirichlet_dynamic_stiffness.py`: interface dynamic stiffness for the
  displacement-driven substructure.
- `neumann_dynamic_stiffness.py`: interface dynamic stiffness for the
  force-driven substructure.
- `fig3_*`, `fig4_*`, `fig5_6_*`, and `fig7_*`: manuscript figure scripts.
